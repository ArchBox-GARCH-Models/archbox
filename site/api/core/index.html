    Core API¶

    VolatilityModel¶

    VolatilityModel ¶

    Bases: ABC

    Abstract base class for volatility models.

    All volatility models (GARCH, EGARCH, GJR-GARCH, etc.) inherit from this class.

    Parameters:

    Name Type Description Default
    endog array - like

    Time series of returns.

    		 required
    mean str

    Mean model: 'constant' (demean) or 'zero'.

    		 'constant'
    dist str

    Conditional distribution: 'normal', 'studentt', 'skewt'.

    		 'normal'

    Attributes:

    Name Type Description
    endog NDArray[float64]

    Returns (demeaned if mean='constant').
    nobs int

    Number of observations.
    dist Distribution

    Conditional distribution instance.
    volatility_process str

    Name of the volatility process.
    mu float

    Estimated mean (0 if mean='zero').

    Fit the model via Maximum Likelihood Estimation.

    Parameters:

    Name Type Description Default
    method str

    Estimation method. Currently only 'mle'.

    		 'mle'
    starting_values ndarray

    Custom starting values. If None, uses self.start_params.

    		 None
    variance_targeting bool

    If True, fix omega = var * (1 - persistence).

    		 False
    disp bool

    Display optimization progress.

    		 True

    Type Description

    Fitted model results.

    Estimated parameters.

    persistence() ¶

    Compute persistence: sum(alpha_i) + sum(beta_j).

    Returns:

    Type Description
    float

    Persistence value. Must be < 1 for stationarity.

    half_life() ¶

    Compute half-life of volatility shocks.

    Returns:

    Type Description
    float

    Number of periods for a shock to decay by half.

    unconditional_variance() ¶

    Compute unconditional (long-run) variance.

    Returns:

    Type Description
    float

    omega / (1 - persistence).

    forecast(horizon=1, method='analytic') ¶

    Forecast conditional variance h steps ahead.

    For GARCH(1,1): E[sigma^2_{T+h}] = sigma^2_inf + (alpha+beta)^{h-1} * (sigma^2_{T+1} - sigma^2_inf)

    Parameters:

    Name Type Description Default
    horizon int

    Number of steps ahead.

    		 1
    method str

    Forecast method: 'analytic'.

    		 'analytic'

    Returns:

    Type Description
    dict

    Dictionary with keys: - 'variance': Forecasted variance, shape (horizon,) - 'volatility': Forecasted volatility (sqrt), shape (horizon,)
