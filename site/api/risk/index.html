    parametric(dist='normal', nu=8.0) ¶

    Compute parametric VaR.

    Parameters:

    Name Type Description Default
    dist str

    Distribution: 'normal' or 'studentt'. Default is 'normal'.

    		 'normal'
    nu float

    Degrees of freedom for Student-t. Default is 8.0.

    		 8.0

    Returns:

    Type Description
    NDArray[float64]

    VaR series, shape (T,). Negative values indicate losses.
    Notes

    Normal: VaR_alpha = mu + sigma_t * Phi^{-1}(alpha)

    Student-t: VaR_alpha = mu + sigma_t * t^{-1}_nu(alpha) * sqrt((nu-2)/nu)

    historical(window=250) ¶

    Compute VaR by Historical Simulation.

    Parameters:

    Name Type Description Default
    window int

    Rolling window size. Default is 250 (approx. 1 year).

    		 250

    Returns:

    Type Description
    NDArray[float64]

    VaR series, shape (T,). First window values are NaN.
    Notes

    VaR_alpha = quantile(r_{t-W+1}, ..., r_t ; alpha)

    filtered_historical() ¶

    Compute VaR by Filtered Historical Simulation (FHS).

    Returns:

    Type Description
    NDArray[float64]

    VaR series, shape (T,).
    Notes

    Barone-Adesi et al. (1999): 1. z_t = (r_t - mu) / sigma_t (standardized residuals) 2. VaR_t = mu + sigma_t * quantile(z_1, ..., z_{t-1} ; alpha)

    The FHS method combines the GARCH volatility dynamics with the empirical distribution of standardized residuals.

    monte_carlo(n_sims=10000, horizon=1, seed=None) ¶

    Compute VaR by Monte Carlo simulation.

    Parameters:

    Name Type Description Default
    n_sims int

    Number of simulation paths. Default is 10000.

    		 10000
    horizon int

    Forecast horizon in periods. Default is 1.

    		 1
    seed int

    Random seed for reproducibility.

    		 None

    Returns:

    Type Description
    NDArray[float64]

    VaR estimate(s). If horizon=1, returns scalar-like array. If horizon>1, returns array of shape (horizon,).
    Notes
    1. Use the last conditional variance as starting point.
    2. Simulate N paths of sigma^2_{T+h} and r_{T+h}.
    3. VaR_alpha = quantile(simulated returns ; alpha).

    parametric(dist='normal', nu=8.0) ¶

    Compute parametric Expected Shortfall.

    Parameters:

    Name Type Description Default
    dist str

    Distribution: 'normal' or 'studentt'. Default is 'normal'.

    		 'normal'
    nu float

    Degrees of freedom for Student-t. Default is 8.0.

    		 8.0

    Returns:

    Type Description
    NDArray[float64]

    ES series, shape (T,). Negative values indicate expected losses.
    Notes

    Normal: ES_alpha = mu - sigma * phi(z_alpha) / alpha

    Student-t: ES_alpha = mu - sigma * (f_nu(t_alpha) / alpha) * ((nu + t_alpha^2) / (nu - 1)) * sqrt((nu-2)/nu)

    		 250

    Returns:

    Type Description

    ES series, shape (T,). First window values are NaN.
    Notes

    ES_alpha = mean(r_t | r_t < VaR_alpha) in rolling window.

    filtered_historical() ¶

    Compute ES by Filtered Historical Simulation.

    Returns:

    Type Description
    NDArray[float64]

    ES series, shape (T,).
    Notes
    1. z_t = (r_t - mu) / sigma_t
    2. ES_t = mu + sigma_t * mean(z_s | z_s < quantile(z; alpha))

    kupiec_test() ¶

    Kupiec (1995) Proportion of Failures (POF) test.

    Tests H0: violation rate = alpha.

    Returns:

    Type Description
    TestResult

    LR_POF statistic and p-value, chi2(1).
    Notes

    LR_POF = -2 * [xlog(alpha) + (n-x)log(1-alpha) - xlog(pi_hat) - (n-x)log(1-pi_hat)] LR_POF ~ chi2(1)

    christoffersen_test() ¶

    Christoffersen (1998) Conditional Coverage test.

    Tests H0: violations are independent AND rate = alpha.

    Returns:

    Type Description
    TestResult

    LR_CC statistic and p-value, chi2(2).
    Notes

    LR_CC = LR_POF + LR_ind ~ chi2(2)

    Where LR_ind tests independence of the hit sequence.
